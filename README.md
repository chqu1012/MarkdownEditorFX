# MarkdownEditorFX
A small markdown editor with some useful features:
* syntax highlighting
* workspace
* markdown preview
* Autocompletion
* templates
* pdf export

Editor features:
* Zoom in / out / reset
* Fold / unfold sections
* Upper / lower / title case letters
* Dark / light theme switch
* Show / hide map
* Show / hide line numbers
* Block comment
